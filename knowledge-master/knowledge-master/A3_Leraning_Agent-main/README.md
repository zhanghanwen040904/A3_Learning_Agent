# A3 Learning Agent

基于大模型的个性化资源生成与学习多智能体系统，面向第十五届中国软件杯 A3 赛题开发。

## 1. 项目简介

A3 Learning Agent 以《人工智能导论》课程为样例课程，融合讯飞星火大模型、SeeDance 多模态生成、RAG 课程知识库、内容安全审核和多智能体协同机制，为高校学生提供个性化学习资源、学习路径、智能答疑和学习效果评估。

## 2. 核心能力

- 对话式学习画像：自然语言生成不少于 6 个维度的动态学生画像；
- 多智能体协同：11 个智能体协作完成画像、检索、资源、路径、答疑、评估和安全复核；
- 多模态资源生成：讲解文档、练习题、拓展阅读、思维导图、代码案例、教学视频/动画任务；
- 个性化学习路径：结合画像、知识短板和掌握度生成阶段化学习计划；
- 智能辅导：基于课程知识库进行答疑，展示课程来源；
- 学习评估闭环：练习作答、自动评分、掌握度记录、画像随学随新；
- 防幻觉机制：RAG 来源引用、关键词降级检索、内容审核、SafetyAgent 复核。

## 3. 技术栈

| 层级 | 技术 |
| --- | --- |
| 前端 | Vue 3、Vite、Element Plus、Vue Router、ECharts、Markdown-It |
| 后端 | Flask、PyMySQL、JWT、Flask-Cors |
| 数据库 | MySQL |
| RAG | ChromaDB、Sentence Transformers、关键词降级检索 |
| AI 能力 | 讯飞 Spark X、讯飞 SeeDance、讯飞内容审核 |

## 5. 快速启动

### 5.1 后端

```bash
cd backend
pip install -r requirements.txt
copy .env.example .env
python app.py
```

### 5.2 前端

```bash
cd frontend
npm install
npm run dev
```

访问：

```text
http://localhost:5173
```

## 6. 数据库初始化

## 8. 推荐演示输入

画像输入：

```text
我是计算机专业大二学生，学过 Python 和高等数学，但人工智能基础比较薄弱，尤其不理解监督学习、无监督学习和神经网络。我希望两周内掌握人工智能导论的核心知识，每天可以学习 1 小时，喜欢通过案例、图解和代码来学习。
```

答疑输入：

```text
请用通俗语言解释监督学习和无监督学习的区别，并给我一个自测题。
```

评估答案：

```text
监督学习使用带标签数据进行训练，目标是学习输入到输出的映射，常见任务包括分类和回归。无监督学习使用无标签数据，主要用于发现数据内部结构，例如聚类和降维。
```

## 10. 注意事项

- 不要提交真实 `.env` 密钥文件；
- 如无真实讯飞 API，可设置 `MOCK_AI=true` 进行稳定演示；
- 首次使用请进入“知识库管理”页面重建知识库；
- 向量模型不可用时系统会自动启用关键词检索降级模式。
