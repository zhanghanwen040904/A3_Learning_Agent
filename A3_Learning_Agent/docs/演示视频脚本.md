# 7分钟演示视频脚本

## 0:00-0:30 项目开场

大家好，我们展示的作品是 A3 Learning Agent，基于大模型的个性化资源生成与学习多智能体系统。本项目面向高校《人工智能导论》课程，解决学生学习资源繁杂、学习路径不清晰和个性化辅导不足的问题。

## 0:30-1:10 多智能体架构展示

进入系统后，首先展示“多智能体架构”页面。这里可以看到系统由 11 个智能体组成，包括画像分析、知识检索、资源生成、路径规划、智能答疑、学习评估和安全复核等角色。系统还展示了与 A3 赛题要求的对应关系。

## 1:10-2:00 对话式学习画像

进入“对话式画像”页面，输入学生情况：

```text
我是计算机专业大二学生，学过 Python 和高等数学，但人工智能基础比较薄弱，尤其不理解监督学习、无监督学习和神经网络。我希望两周内掌握人工智能导论的核心知识，每天可以学习 1 小时，喜欢通过案例、图解和代码来学习。
```

系统自动抽取知识基础、学习风格、薄弱点、学习目标、时间偏好和课程进度，形成动态学生画像。

## 2:00-2:50 知识库管理与 RAG 检索

进入“知识库管理”页面。系统已经构建了《人工智能导论》10 章课程文档，包括人工智能概述、机器学习基础、监督学习、无监督学习、神经网络等内容。点击重建知识库后，系统会显示当前检索模式。输入“监督学习和无监督学习有什么区别”，可以看到返回的课程来源片段。

## 2:50-3:50 多智能体资源生成

进入“学习资源”页面，点击生成学习资源。系统会通过多个智能体协同生成课程讲解文档、练习题、拓展阅读、思维导图、代码实操案例和教学视频/动画任务。每类资源都围绕学生画像中的薄弱点生成。

## 3:50-4:35 个性化学习路径

进入“学习路径”页面，点击生成学习路径。系统会结合画像、课程知识库和掌握度，输出阶段化学习计划，包括每一步学习目标、资源推荐、练习方式和评估指标。

## 4:35-5:35 智能答疑与防幻觉

进入“智能答疑”页面，输入：

```text
请用通俗语言解释监督学习和无监督学习的区别，并给我一个自测题。
```

系统先检索课程知识库，再以伪流式方式生成回答，并展示参考来源标签。这体现了系统的 RAG 防幻觉机制，回答不是单纯依赖大模型自由生成，而是有课程依据。

## 5:35-6:25 学习效果评估

进入“学习评估”页面，提交练习答案：

```text
监督学习使用带标签数据进行训练，目标是学习输入到输出的映射，常见任务包括分类和回归。无监督学习使用无标签数据，主要用于发现数据内部结构，例如聚类和降维。
```

系统自动评分，更新知识点掌握度，并生成下一步学习任务。评估结果还会反向更新学生画像，实现画像随学随新。

## 6:25-7:00 总结

本系统完整覆盖 A3 赛题要求：对话式画像、多智能体资源生成、个性化路径、智能答疑、学习评估、防幻觉和内容安全。创新点在于将多智能体、RAG、多模态生成和学习闭环结合，使大模型真正服务于高校课程的个性化学习。
