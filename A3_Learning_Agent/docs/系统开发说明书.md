# 系统开发说明书

## 系统结构

- 前端负责会话、画像、路径、评估与知识库页面展示
- 后端负责业务接口、数据库访问、RAG 检索与智能体协同
- 大模型负责对话回答、画像抽取、资源生成等智能能力

## 当前模型方案

项目已统一切换为讯飞星火 Spark Ultra-32K HTTP 兼容接口。

## 关键模块

- `backend/ai/llm_api.py`：统一大模型调用入口
- `backend/ai/llm_adapter.py`：LangChain 适配层
- `backend/ai/agents/`：多智能体实现
- `backend/api/`：后端接口

## 前端重点页面

- 对话页
- 学生画像页
- 学习路径页
- 学习评估页
- 知识库管理页
- 系统状态页
