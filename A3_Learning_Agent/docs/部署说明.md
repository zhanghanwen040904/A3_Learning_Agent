# 部署说明

## 1. 后端

```bash
cd backend
pip install -r requirements.txt
copy .env.example .env
python app.py
```

## 2. 前端

```bash
cd frontend
npm install
npm run dev
```

## 3. 讯飞星火配置

```env
AI_PROVIDER=spark
AI_MAX_INPUT_CHARS=24000
AI_MAX_TOKENS=4096
SPARK_APIPASSWORD=你的讯飞 APIPassword
SPARK_BASE_URL=https://spark-api-open.xf-yun.com/v1/chat/completions
SPARK_MODEL=4.0Ultra
```

## 4. 演示模式

```env
MOCK_AI=true
```

## 5. 注意事项

- 不要提交真实 `.env`
- 前端构建产物建议按需生成，不长期保留
- 变更知识库后需要重建索引
