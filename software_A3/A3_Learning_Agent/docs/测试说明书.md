# 测试说明书

## 1. 后端启动测试

测试目标：确认 Flask 服务可正常启动。

测试命令：

```bash
python app.py
curl http://localhost:5000/health
```

预期结果：

```json
{"code":200,"msg":"成功","data":{"status":"ok","service":"A3 Learning Agent"}}
```

## 2. 数据库连接测试

测试目标：确认 MySQL 连接、插入、查询正常。

测试命令：

```bash
python -c "from db import mysql_db; print(mysql_db.query_one('SELECT 1 AS ok'))"
```

预期结果：

```json
{"ok": 1}
```

## 3. 讯飞接口封装测试

测试目标：确认讯飞星火、SeeDance、内容审核封装函数可调用。

测试命令：

```bash
python -c "from ai.spark_api import spark_chat; print(spark_chat('用一句话解释人工智能'))"
python -c "from ai.spark_api import content_audit; print(content_audit('人工智能导论课程学习'))"
```

说明：SeeDance 需配置真实接口地址后测试。

## 4. RAG 检索测试

测试目标：确认课程文档可被向量化并检索。

测试步骤：

1. 将《人工智能导论》txt/pdf/docx 资料放入 `rag_data/source_docs`。
2. 执行：

```bash
python -c "from ai.rag import build_vector_db, retrieve_knowledge; print(build_vector_db()); print(retrieve_knowledge('监督学习', 3))"
```

预期结果：返回包含“教材来源”和原文片段的文本。

## 5. 多智能体流水线测试

测试目标：确认画像、检索、文本、可视化、视频智能体可被统一调度。

测试命令：

```bash
python -c "from ai.agents import agent_manager; print(agent_manager.run_pipeline('我机器学习基础一般，想学习监督学习，喜欢代码案例，晚上学习。'))"
```

预期结果：返回 profile、knowledge、resource_list、errors 字段。

## 6. 业务接口测试

推荐使用 Postman 或 curl 测试以下接口：

- `POST /api/auth/register`
- `POST /api/auth/login`
- `POST /api/profile/create`
- `POST /api/profile/update`
- `POST /api/resource/generate`
- `POST /api/path/generate`
- `POST /api/chat/answer`

所有接口统一返回：

```json
{"code":200,"msg":"成功","data":{}}
```

## 7. 前端联调测试

测试目标：确认页面可访问、接口可调用、Markdown 渲染和进度条可显示。

测试步骤：

1. 启动后端 `python app.py`。
2. 启动前端 `npm run dev`。
3. 访问 `http://localhost:5173`。
4. 依次测试登录注册、画像构建、资源生成、路径生成、智能答疑。
