# PPT 大纲

## 1. 项目背景

- 面向高校课程学习场景
- 目标：构建多智能体协同学习辅助平台

## 2. 核心能力

- 对话式学习辅助
- 学生画像自动构建与更新
- 多类型学习资源生成
- 学习路径规划
- 学习评估与反馈闭环

## 3. 技术架构

- 前端：Vue 3
- 后端：Flask
- 数据库：MySQL
- 检索：RAG + 降级关键词检索
- 大模型：阿里云百炼兼容接口

## 4. 多智能体协同

- ProfileAgent：画像更新
- RetrieveAgent：知识检索
- PlannerAgent：任务规划
- TutorAgent：对话答疑
- EvaluatorAgent：评估反馈

## 5. 比赛要求对应

- 多智能体架构：已体现
- 多形式资源：文档 / 题目 / 阅读 / 导图 / 代码 / 视频脚本
- 个性化：由画像、知识库、学习记录共同驱动
