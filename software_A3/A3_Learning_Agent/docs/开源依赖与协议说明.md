# 开源依赖与协议说明

本项目遵循第十五届中国软件杯 A3 赛题要求，所有 AI 生成能力仅对接科大讯飞星火 V3.5、讯飞 SeeDance 和讯飞内容审核 API。以下为项目使用的主要开源依赖及常见协议说明，最终提交前建议以各项目官方仓库 LICENSE 为准再次核验。

## AI 服务说明

- 讯飞星火 V3.5：用于画像抽取、文本资源生成、学习路径规划、智能答疑。
- 讯飞 SeeDance：用于教学短视频生成。
- 讯飞内容审核 API：用于输入与输出内容安全过滤。

## AI Coding 工具说明

项目开发过程中使用 AI Coding 辅助工具进行代码生成、重构和文档编写，所有生成内容均围绕本项目从零开发，不直接复用外部项目源码。
