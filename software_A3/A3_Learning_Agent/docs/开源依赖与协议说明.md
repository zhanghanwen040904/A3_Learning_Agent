本项目为第十五届中国软件杯 A3 赛题作品。项目使用开源框架完成 Web 应用、RAG 检索、数据库访问、前端 UI 和 Markdown 渲染等功能。所有第三方依赖仅用于学习、开发和比赛演示，最终提交时应保留本说明。

## 5. DeepTutor 参考说明

项目开发过程中参考了 DeepTutor 一类智能教育系统的设计思想，例如：

- 学习画像；
- 课程知识库；
- 掌握度追踪；
- 智能体协同。

本项目为独立实现，未直接复制外部项目核心源码。如后续引入 DeepTutor 具体代码，应根据其官方仓库协议保留来源与协议声明。

## 6. 许可证合规要求

1. 不删除第三方依赖自带 LICENSE；
2. 不将第三方开源组件声明为自研；
3. 提交作品时保留本说明文档；
4. 若使用真实 API 密钥，不得提交 `.env` 文件；
5. 可提交 `.env.example` 作为环境模板。
