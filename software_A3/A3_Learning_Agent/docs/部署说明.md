# 部署说明

## 1. 项目路径

项目根目录：

```bash
D:\软件\pycharm_professional‌\my_pythonProgram\software_A3\A3_Learning_Agent
```

## 2. 后端启动

进入后端目录：

```bash
cd D:\软件\pycharm_professional‌\my_pythonProgram\software_A3\A3_Learning_Agent\backend
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

健康检查：

```bash
curl http://localhost:5000/health
```

## 3. 后端环境变量

在 `backend/.env` 中配置：

```env
APP_PORT=5000
MYSQL_HOST=127.0.0.1
MYSQL_PORT=3306
MYSQL_USER=root
MYSQL_PASSWORD=你的MySQL密码
MYSQL_DATABASE=a3_learning_agent

XFYUN_APP_ID=你的讯飞APP_ID
XFYUN_API_KEY=你的讯飞API_KEY
XFYUN_API_SECRET=你的讯飞API_SECRET
XFYUN_SPARK_URL=wss://spark-api.xf-yun.com/v3.5/chat
XFYUN_SPARK_DOMAIN=generalv3.5

SEEDANCE_API_KEY=你的SeeDance密钥
SEEDANCE_API_URL=你的SeeDance接口地址
CONTENT_AUDIT_API_KEY=你的内容审核密钥
CONTENT_AUDIT_API_URL=你的内容审核接口地址
```

## 4. 数据库初始化

```bash
mysql -u root -p < backend/db/init.sql
```

初始化后包含 4 张核心表：

- `user`
- `student_profile`
- `study_resource`
- `study_path`

## 5. RAG 知识库初始化

把《人工智能导论》课程资料放入：

```bash
rag_data/source_docs
```

支持格式：

- `.txt`
- `.pdf`
- `.docx`

首次调用检索时会自动构建 Chroma 向量库，并持久化到：

```bash
rag_data/vector_db
```

也可手动执行：

```bash
cd backend
python -c "from ai.rag import build_vector_db; print(build_vector_db())"
```

## 6. 前端启动

```bash
cd D:\软件\pycharm_professional‌\my_pythonProgram\software_A3\A3_Learning_Agent\frontend
npm install
npm run dev
```

浏览器访问：

```text
http://localhost:5173
```

## 7. 核心接口测试

### 注册

```bash
curl -X POST http://localhost:5000/api/auth/register -H "Content-Type: application/json" -d "{\"username\":\"test\",\"password\":\"123456\"}"
```

### 登录

```bash
curl -X POST http://localhost:5000/api/auth/login -H "Content-Type: application/json" -d "{\"username\":\"test\",\"password\":\"123456\"}"
```

### 创建画像

```bash
curl -X POST http://localhost:5000/api/profile/create -H "Content-Type: application/json" -d "{\"user_id\":1,\"dialogue\":\"我是人工智能专业学生，机器学习基础一般，想补齐监督学习和无监督学习，喜欢案例和代码，晚上学习。\"}"
```

### 生成资源

```bash
curl -X POST http://localhost:5000/api/resource/generate -H "Content-Type: application/json" -d "{\"user_id\":1}"
```

### 生成学习路径

```bash
curl -X POST http://localhost:5000/api/path/generate -H "Content-Type: application/json" -d "{\"user_id\":1}"
```

### 智能答疑

```bash
curl -X POST http://localhost:5000/api/chat/answer -H "Content-Type: application/json" -d "{\"question\":\"监督学习和无监督学习有什么区别？\",\"need_video\":false}"
```
