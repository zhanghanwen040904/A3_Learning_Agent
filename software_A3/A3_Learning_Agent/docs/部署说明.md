# 部署说明

## 1. 后端

```bash
cd backend
pip install -r requirements.txt
copy .env.example .env
python app.py
```

## 2. 前端

```bash
cd frontend
npm install
npm run dev
```

## 3. 百炼配置

```env
AI_PROVIDER=bailian
BAILIAN_API_KEY=你的百炼 API Key
BAILIAN_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions
BAILIAN_MODEL=qwen-plus
```

## 4. 演示模式

```env
MOCK_AI=true
```

## 5. 注意事项

- 不要提交真实 `.env`
- 前端构建产物建议按需生成，不长期保留
- 变更知识库后需要重建索引
