# 需求规格说明书

## 1. 项目概述

A3 Learning Agent 是面向个性化学习场景的多智能体系统，支持学习者画像解析、课程知识检索、文本资源生成、可视化资源生成、多模态视频生成和智能答疑。

## 2. 用户角色

- 学习者：获取个性化学习路径、学习资源和答疑服务。
- 教师/管理员：维护课程资料、查看学习资源生成结果。

## 3. 核心功能

- 学习画像分析
- RAG 知识库检索
- 个性化学习路径生成
- 文本、可视化、视频资源生成
- 智能问答与内容审核

## 4. 非功能需求

- 后端接口支持跨域访问。
- 密钥和数据库配置通过环境变量管理。
- 数据库脚本支持初始化核心业务表。
