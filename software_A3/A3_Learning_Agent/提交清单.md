## 3. 提交前检查

### 后端

```bash
cd backend
python -m compileall .
python app.py
```

### 前端

```bash
cd frontend
npm run build
npm run dev
```

### 页面流程

```text
登录/注册 → 多智能体架构 → 对话式画像 → 知识库管理 → 学习资源 → 学习路径 → 智能答疑 → 学习评估
```
