# 第3章 知识表示与推理

## 学习目标
理解知识表示的作用，掌握命题逻辑、谓词逻辑、产生式规则和语义网络的基本思想。

## 核心概念
知识表示用于把现实世界中的事实、概念和关系转换为机器可处理的形式。推理是在已有知识基础上推出新结论的过程。

## 典型方法
命题逻辑用真假命题描述事实，谓词逻辑能表达对象、属性和关系。产生式规则通常写作 IF 条件 THEN 结论，适合专家系统。语义网络使用节点和边表示概念及关系。

## 学习重点
1. 理解知识表示和机器学习的区别。
2. 掌握 IF-THEN 规则表达方式。
3. 理解推理过程中前向推理和后向推理的区别。

## 易错点
知识表示强调显式表达知识，机器学习强调从数据中学习模式。两者可以结合，而不是互相替代。

## 练习题
1. 什么是知识表示？
2. 用产生式规则描述“如果下雨则带伞”。
3. 前向推理和后向推理有什么不同？
