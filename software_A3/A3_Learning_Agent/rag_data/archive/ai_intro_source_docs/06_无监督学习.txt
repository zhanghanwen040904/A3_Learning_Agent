# 第6章 无监督学习

## 学习目标
理解无监督学习的基本思想，掌握聚类、降维和关联分析的典型应用。

## 核心概念
无监督学习处理没有标签的数据，目标是发现样本之间的结构、相似性或潜在模式。它常用于数据探索、用户分群、异常检测和特征压缩。

## 聚类
聚类把相似样本划分到同一组。K-Means 是常见聚类算法，需要预先指定簇的数量 K，通过不断更新簇中心来最小化样本到中心的距离。

## 降维
降维把高维数据映射到低维空间，减少冗余特征和计算成本。主成分分析 PCA 是典型降维方法，寻找能最大保留数据方差信息的方向。

## 学习重点
1. 理解无标签数据和有标签数据的区别。
2. 掌握聚类和分类的区别。
3. 了解降维在可视化和特征压缩中的作用。

## 易错点
聚类不是分类。分类需要已知标签训练模型，聚类是在没有标签的情况下自动发现群组。

## 练习题
1. K-Means 为什么需要指定 K？
2. 聚类和分类有什么区别？
3. PCA 的主要作用是什么？
