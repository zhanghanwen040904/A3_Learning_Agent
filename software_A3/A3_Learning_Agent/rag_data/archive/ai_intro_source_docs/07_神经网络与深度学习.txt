# 第7章 神经网络与深度学习

## 学习目标
理解神经网络的基本结构，掌握感知机、多层神经网络、激活函数、反向传播和深度学习应用。

## 核心概念
神经网络由大量人工神经元连接而成，通过权重和偏置对输入进行变换。多层神经网络能够学习非线性关系。深度学习通常指包含多层网络结构的学习方法。

## 训练过程
神经网络训练通常包括前向传播、损失计算、反向传播和参数更新。损失函数衡量预测结果与真实标签之间的差距，优化算法用于调整模型参数。

## 常见结构
卷积神经网络适合图像处理，循环神经网络适合序列数据，Transformer 结构在自然语言处理和多模态任务中广泛使用。

## 学习重点
1. 理解神经元、权重、偏置和激活函数。
2. 掌握损失函数和反向传播的作用。
3. 了解深度学习与传统机器学习的区别。

## 易错点
神经网络不是层数越多越好。过深的网络可能带来过拟合、训练困难和算力成本增加。

## 练习题
1. 激活函数的作用是什么？
2. 反向传播用于解决什么问题？
3. CNN 为什么适合图像任务？
