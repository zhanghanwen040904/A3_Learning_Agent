# 第5章 监督学习

## 学习目标
掌握监督学习的基本思想，理解分类、回归、决策树、线性模型和模型评估。

## 核心概念
监督学习的数据样本包含输入特征和目标标签。模型通过训练样本学习输入到标签之间的映射关系，并用于预测新样本。

## 分类任务
分类任务的输出是离散类别。例如垃圾邮件识别、鸢尾花类别预测和图像类别识别。常见分类算法包括决策树、朴素贝叶斯、支持向量机和神经网络。

## 回归任务
回归任务的输出是连续数值。例如房价预测、温度预测和销量预测。常见回归方法包括线性回归、岭回归和回归树。

## 模型评估
分类任务需要关注准确率、精确率、召回率和混淆矩阵。类别不均衡时只看准确率可能产生误导，应结合召回率和 F1 值分析。

## 学习重点
1. 判断问题是分类还是回归。
2. 理解决策树的可解释性优势。
3. 掌握训练集和测试集划分的重要性。

## 易错点
房价预测属于回归，不属于分类；垃圾邮件识别属于分类，不属于回归。

## 练习题
1. 鸢尾花类别预测属于什么任务？
2. 房价预测属于什么任务？
3. 为什么类别不均衡时不能只看准确率？
