# 第2章 搜索与问题求解

## 学习目标
理解问题求解的基本形式化方法，掌握状态空间、初始状态、目标状态、算子和路径代价等概念。

## 核心概念
搜索是人工智能早期的重要方法，用于在可能状态空间中寻找从初始状态到目标状态的路径。典型搜索方法包括广度优先搜索、深度优先搜索、一致代价搜索和启发式搜索。

## 启发式搜索
启发式搜索利用启发函数估计当前状态到目标状态的距离，从而减少搜索空间。A* 算法综合考虑已走代价和预计剩余代价，常用评价函数 f(n)=g(n)+h(n)。

## 学习重点
1. 能将实际问题转化为状态空间问题。
2. 理解盲目搜索和启发式搜索的区别。
3. 掌握 A* 算法中 g、h、f 的含义。

## 易错点
启发式函数不是最终答案，只是对目标距离的估计。启发函数设计不合理可能导致搜索效率低或路径不理想。

## 练习题
1. 什么是状态空间？
2. 广度优先搜索和深度优先搜索有什么区别？
3. A* 算法为什么通常比盲目搜索更高效？
