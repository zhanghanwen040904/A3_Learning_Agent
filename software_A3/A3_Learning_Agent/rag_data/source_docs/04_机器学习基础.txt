# 第4章 机器学习基础

## 学习目标
理解机器学习的基本定义、任务类型、训练流程和评价方法。

## 核心概念
机器学习是让计算机系统利用数据和经验自动改进性能的方法。一个典型机器学习流程包括数据收集、数据预处理、特征构建、模型训练、模型评估和部署应用。

## 任务类型
监督学习使用带标签数据训练模型，目标是学习输入到输出的映射，常见任务包括分类和回归。无监督学习使用无标签数据，目标是发现数据内部结构，常见任务包括聚类和降维。强化学习通过智能体与环境交互，根据奖励信号学习策略。

## 评价指标
分类任务常用准确率、召回率、精确率和 F1 值。回归任务常用均方误差和平均绝对误差。聚类任务可使用轮廓系数等指标。

## 学习重点
1. 区分监督学习、无监督学习和强化学习。
2. 理解训练集、验证集和测试集的作用。
3. 了解过拟合与欠拟合。

## 易错点
有标签并不等于数据质量高；标签错误会影响监督学习效果。测试集不能参与模型训练，否则评估结果会过于乐观。

## 练习题
1. 什么是监督学习？
2. 分类和回归的输出有什么不同？
3. 什么是过拟合？
