# 第9章 计算机视觉

## 学习目标
理解计算机视觉的目标、基本任务和常用深度学习方法。

## 核心概念
计算机视觉研究如何让机器从图像和视频中获取信息。典型任务包括图像分类、目标检测、图像分割、人脸识别和视频理解。

## 图像分类
图像分类判断整张图像属于哪个类别。卷积神经网络通过卷积核提取局部特征，再通过多层网络组合成高级语义特征。

## 目标检测与分割
目标检测不仅判断类别，还需要定位目标位置。图像分割需要把图像中的像素划分到不同类别，适合医学影像和自动驾驶等场景。

## 学习重点
1. 区分图像分类、目标检测和图像分割。
2. 理解卷积操作提取局部特征的思想。
3. 了解计算机视觉在教育、医疗和交通中的应用。

## 易错点
图像分类只输出类别，目标检测还输出位置，图像分割进一步输出像素级区域。

## 练习题
1. 图像分类和目标检测有什么区别？
2. 卷积神经网络为什么适合图像任务？
3. 图像分割适合哪些场景？
