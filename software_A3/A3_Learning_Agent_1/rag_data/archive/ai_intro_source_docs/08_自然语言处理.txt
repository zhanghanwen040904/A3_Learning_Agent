# 第8章 自然语言处理

## 学习目标
理解自然语言处理的目标、基本任务和大语言模型的基本能力。

## 核心概念
自然语言处理研究如何让计算机理解、生成和处理人类语言。典型任务包括分词、词性标注、文本分类、信息抽取、机器翻译、问答系统和文本生成。

## 表示学习
早期方法依赖词袋模型和 TF-IDF，现代方法使用词向量和预训练语言模型。词向量把词映射为连续向量，使语义相近的词在向量空间中更接近。

## 大语言模型
大语言模型通过大规模语料预训练获得语言理解和生成能力。它可以用于智能问答、摘要生成、代码辅助、学习资源生成等任务，但也可能产生幻觉，需要 RAG 和内容审核约束。

## 学习重点
1. 理解文本分类、信息抽取和问答系统的区别。
2. 了解词向量和预训练模型的作用。
3. 认识大语言模型幻觉问题。

## 易错点
大语言模型生成的内容不一定都正确，必须结合可信知识库、引用来源和安全审核机制。

## 练习题
1. 什么是自然语言处理？
2. 词向量有什么作用？
3. 为什么大语言模型需要 RAG 增强？
