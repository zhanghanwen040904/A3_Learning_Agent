### 程序的潜在错误

起始页：130
[第130页] Halstead度量还可以用来预测程序中可能存在的错 误E。预测公式为
[第130页] E = (N1+N2)log2(n1+n2) / 3000
[第130页] 例如，一个程序对75个数据库项共访问1300次， 对150个运算符共使用了1200次，那么预测该程序 的错误数： E = (1300+1200)log2(150+75)/3000  6.5 即预测该程序中可能包含6～7个错误
[第130页] 实验数据表明，预测的错误数量与实际错误数量的误差在8%之内
