# 需求规格说明书

## 1. 赛题背景

在数字化与智能化深度融合背景下，高等教育正在从统一化教学走向个性化学习。不同学生在专业背景、知识基础、学习目标、认知风格、时间安排和实践能力上差异明显，传统课堂与通用学习平台难以持续提供精准资源和动态学习指导。

A3 Learning Agent 面向 A3 赛题“基于大模型的个性化资源生成与学习多智能体系统开发”，以《人工智能导论》课程为切入点，融合大语言模型、RAG 课程知识库、多智能体协同、多模态生成、内容安全审核和学习效果评估，构建可运行的个性化学习智能体系统。

## 2. 真实业务痛点

### 2.1 学习资源繁杂无序
高校学生面对教材、课件、题库、视频、论文和代码仓库时，常常不知道哪些资源适合当前水平。

### 2.2 标准化教学难以因材施教
课堂教学通常面向整体学生，无法根据每位学生的薄弱点和学习节奏动态调整内容。

### 2.3 学习反馈滞后
传统作业和考试反馈周期较长，学生无法及时了解知识掌握情况。

### 2.4 AI 生成内容存在幻觉风险
通用大模型可能生成看似合理但事实错误的学术内容，需要可信课程知识库和安全复核机制。

## 3. 系统目标

系统目标是构建一个面向高校课程学习的多智能体系统，实现：

1. 通过自然语言对话自动构建动态学生画像；
2. 基于课程知识库和 RAG 生成可信学习内容；
3. 多智能体协同生成不少于 5 类个性化学习资源；
4. 结合画像、学习进度和掌握度生成动态学习路径；
5. 提供即时智能答疑和多模态讲解；
6. 通过练习作答、评分反馈和掌握度记录形成学习闭环；
7. 对生成内容进行来源展示、内容审核和防幻觉复核。

## 4. 用户角色

| 角色 | 描述 | 核心需求 |
| --- | --- | --- |
| 学生 | 高校课程学习者 | 获取个性化资源、路径、答疑和反馈 |
| 教师 | 课程资源维护者 | 维护课程知识库、查看资源与评估效果 |
| 管理员 | 系统维护人员 | 管理知识库、部署环境和演示数据 |

## 5. 功能需求

### 5.1 对话式学习画像自主构建

系统应支持学生通过自然语言描述专业、基础、目标、薄弱点、学习偏好和学习历史，自动抽取画像。

画像维度不少于 6 个：

1. 知识基础；
2. 认知/学习风格；
3. 易错点或薄弱知识点；
4. 学习目标；
5. 时间偏好；
6. 课程进度。

系统应支持画像随学习行为和练习评估动态更新。

### 5.2 多智能体协同资源生成

系统须体现多智能体架构，由不同角色智能体协作完成资源生成。至少生成 5 类资源，本项目实现 6 类：

1. 课程讲解文档；
2. 知识点思维导图；
3. 不同类型练习题；
4. 拓展阅读材料；
5. 多模态教学视频/动画任务；
6. 代码类实操案例。

### 5.3 个性化学习路径规划和资源推送

系统应根据学生画像、课程知识点、学习进度、评估结果和资源类型生成阶段式学习路径，明确学习顺序、资源推荐、练习任务和评估指标。

### 5.4 智能辅导

系统应提供即时课程答疑服务。答疑输出包括：

- 文字解释；
- 图解说明；
- 自测题；
- 可选短视频讲解任务；
- 教材来源引用。

### 5.5 学习效果评估

系统应记录学生练习作答和学习行为，自动生成评分、反馈、掌握度记录和下一步学习建议，并动态更新画像。

### 5.6 知识库管理

系统应提供《人工智能导论》课程文档集，支持知识库状态查看、重建、检索测试。向量模型不可用时，系统应自动降级为关键词检索，保障演示稳定性。

## 6. 非功能需求

### 6.1 交互体验

- 界面简洁美观，符合现代 AI 产品风格；
- 支持 Markdown 渲染；
- 多模态资源卡片化展示；
- 长任务具备进度条、步骤条或伪流式输出；
- 登录、错误提示和权限控制清晰。

### 6.2 安全性

- 使用 JWT 鉴权；
- 输入和输出接入内容审核；
- RAG 返回课程来源；
- SafetyAgent 进行安全与防幻觉复核；
- 配置文件和密钥通过环境变量管理。

### 6.3 性能与稳定性

- 常规接口响应在可接受范围内；
- 多模态生成提供进度提示，避免白屏等待；
- RAG 支持向量检索和关键词降级检索；
- MOCK_AI 模式支持无真实 AI 密钥时稳定演示。

## 7. 课程知识库要求

系统自建《人工智能导论》课程文档集，包含 10 个章节：

1. 人工智能概述；
2. 搜索与问题求解；
3. 知识表示与推理；
4. 机器学习基础；
5. 监督学习；
6. 无监督学习；
7. 神经网络与深度学习；
8. 自然语言处理；
9. 计算机视觉；
10. 人工智能伦理与安全。

## 8. 验收标准

| 模块 | 验收点 |
| --- | --- |
| 登录注册 | 未登录不可访问业务页面，登录后可正常使用 |
| 学习画像 | 自然语言输入后生成 6 维画像 |
| 知识库 | 能查看文档、重建知识库、检索课程片段 |
| 资源生成 | 至少生成 5 类资源，本项目生成 6 类 |
| 学习路径 | 输出阶段化学习路径和资源推荐 |
| 智能答疑 | 输出回答、自测题和引用来源 |
| 学习评估 | 提交答案后生成评分、反馈和掌握度 |
| 防幻觉 | 输出内容带课程依据和安全复核结果 |
| 文档材料 | 具备需求、开发、测试、部署、协议、AI 工具、PPT、演示脚本 |
