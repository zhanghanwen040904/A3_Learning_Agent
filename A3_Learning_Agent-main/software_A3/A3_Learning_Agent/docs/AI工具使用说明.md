# AI Coding 工具使用说明

本项目开发过程中使用了 AI 编程辅助工具进行需求分析、代码生成、错误定位、依赖兼容处理、前后端联调和文档整理。

## 使用方式
1. 根据 A3 赛题要求拆解系统模块。
2. 辅助生成 Flask API、Vue 页面和多智能体代码。
3. 协助定位依赖安装、接口报错、前端构建等问题。
4. 辅助整理系统开发说明、测试说明、PPT 大纲和演示脚本。

## 人工审核
所有生成代码均经过人工运行、测试和调整。涉及用户登录、JWT 鉴权、数据库结构、AI 接口调用和内容安全逻辑均由开发者结合项目实际情况进行确认。

## 开源项目说明
项目借鉴 DeepTutor 的能力插件化、知识库管理和学习掌握度模型思想，但未直接复制其核心实现。DeepTutor 使用 Apache License 2.0，若后续引用其代码，需要在提交材料中保留协议说明和来源标注。
