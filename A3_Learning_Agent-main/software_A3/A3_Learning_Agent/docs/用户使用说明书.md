# 用户使用说明书

## 1. 登录注册
打开系统后进入登录页。新用户点击立即注册，填写用户名、密码和确认密码。登录成功后系统自动进入对话式画像页面。

## 2. 对话式画像
在输入框中用自然语言描述自己的专业、基础、目标、薄弱点和时间偏好，点击生成 6 维画像。系统会生成知识基础、学习风格、薄弱点、学习目标、时间偏好和课程进度。

## 3. 学习资源
点击生成学习资源，系统会基于画像和课程知识库生成课程讲解文档、练习题、拓展阅读、思维导图、代码案例和教学短视频链接。

## 4. 学习路径
点击生成学习路径，系统会根据学生画像、课程资料和学习评估生成阶段式学习计划。

## 5. 智能答疑
输入人工智能导论课程问题，系统会先检索课程知识库，再生成文字解释、图解说明和自测题，并显示参考来源。

## 6. 学习评估
在学习评估页面提交练习答案，系统会自动评分，更新知识点掌握度，并给出下一步学习任务。

## 7. 知识库管理
知识库管理页面可以查看课程文档、重建向量库和测试知识检索效果。

## 8. 退出登录
点击侧边栏底部退出登录，系统清除登录状态并返回登录页。
